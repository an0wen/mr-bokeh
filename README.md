# mr-bokeh
Mass-Radius diagram of exoplanets, rendered in bokeh with filtering through sliders

Install the package using:
```
pip install mrbokeh
```
In python, import the package and create the figure with:
```
import mrbokeh
mrbokeh.make_plot()
```
By default, the tool creates the figure under "./mrbokeh.html", but you can provide to the function the path with the filename where the figure should be created.
